# Language Diary — PRD (Product Requirements Document)

## 1. 개요
일기를 모국어로 말하면 AI가 영어로 번역하고, 핵심 단어/구문을 학습 포인트로 제공하며, 따라 말하기(발음 평가)로 학습을 완성하는 언어 학습 앱.

## 2. 타겟 유저
- 영어를 배우려는 한국어 사용자
- 모바일 + 태블릿 사용자
- Phase 2에서 다국어 확장

## 3. 기술 스택

- **앱:** React Native (iOS + Android)
- **Backend:** Python FastAPI
- **AI:** OpenAI API (번역, 학습 포인트 생성)
- **DB:** PostgreSQL
- **STT:** OpenAI Whisper API
- **TTS:** ElevenLabs
- **발음 평가:** Azure Speech SDK
- **배포:** 개인 서버

## 4. 핵심 기능

### MVP (Phase 1 — 한국어 → 영어)

| # | 기능 | 설명 |
|---|------|------|
| F1 | 모국어 일기 입력 | STT로 한국어 음성 → 텍스트 변환 |
| F2 | AI 영어 번역 | OpenAI API로 자연스러운 영어 번역 |
| F3 | 학습 포인트 | 중요 단어 + 회화용 구문 추출 및 해설 |
| F4 | 모범 발음 | TTS로 영어 번역문/구문 발음 재생 |
| F5 | 따라 말하기 | 사용자 발음 녹음 + 발음 평가(점수/피드백) |
| F6 | 일기 히스토리 | 날짜별 일기 + 학습 기록 보기 |

## 5. 학습 포인트 정책

### 구문 추출
일기 번역문에서 두 가지를 모두 추출:
- **회화 표현** — 실생활에서 자주 쓰는 표현 (예: "grab lunch with", "end up ~ing")
- **문법 패턴** — 핵심 영어 구조 (예: "I used to ~", "I'm about to ~")

각 구문에 포함할 정보:
- 영어 구문
- 한국어 뜻
- 예문 (일기 문맥 활용)
- 난이도 (CEFR 기준)

### 단어 추출
- **CEFR 등급별 빈도 기반** (A1~C2)
- 고빈도 단어 우선 추출
- MVP: 사용자 레벨 없이 전체 추출 → 난이도 태그만 표시
- Phase 2: 레벨 테스트 결과에 따라 사용자 수준 대비 학습 가치 높은 단어 우선 추천

각 단어에 포함할 정보:
- 영어 단어
- 한국어 뜻
- 품사
- CEFR 등급 (A1~C2)
- 예문 (일기 문맥 활용)

### CEFR 레벨 체계 (Phase 2 연동)
| 레벨 | 설명 |
|------|------|
| A1 | 입문 — 기초 인사, 자기소개 |
| A2 | 초급 — 일상 대화, 간단한 설명 |
| B1 | 중급 — 여행, 업무 기본 소통 |
| B2 | 중상급 — 자유로운 대화, 의견 표현 |
| C1 | 고급 — 복잡한 주제, 유창한 표현 |
| C2 | 최고급 — 원어민 수준 |

## 6. 사용자 플로우 (MVP)

```
[오늘의 일기] → 한국어로 말하기 (STT)
    → [AI 번역] 영어 번역문 표시
    → [학습 포인트] 중요 단어 + 구문 해설
    → [모범 발음] TTS 재생
    → [따라 말하기] 녹음 + 발음 평가
    → [저장]
```

## 7. 화면 구조 (MVP)

```
1. 홈
   - 오늘의 일기 작성 버튼
   - 최근 일기 목록

2. 일기 작성
   - 마이크 버튼 (STT 입력)
   - 한국어 텍스트 표시 (수정 가능)
   - [번역하기] 버튼

3. 학습 화면
   - 영어 번역문 (전체)
   - 학습 포인트 카드 (단어/구문별)
     - 영어 + 한국어 뜻
     - 예문
     - [발음 듣기] TTS 버튼
     - [따라 말하기] 녹음 버튼 → 발음 점수

4. 히스토리
   - 날짜별 일기 목록
   - 일기 상세 (한국어 + 영어 + 학습 포인트)
```

## 8. API 설계 (초안)

```
POST   /api/diary              — 일기 생성 (한국어 텍스트)
POST   /api/diary/{id}/translate — AI 번역 + 학습 포인트 생성
POST   /api/speech/stt          — 음성 → 텍스트
POST   /api/speech/tts          — 텍스트 → 음성
POST   /api/speech/evaluate     — 발음 평가 (음성 + 정답 텍스트)
GET    /api/diary               — 일기 목록
GET    /api/diary/{id}          — 일기 상세
```

## 9. MVP 제외 사항
- 복습 기능
- 레벨 테스트

- 통계 대시보드
- 소셜 공유
- 오프라인 모드

## 10. 리스크 & 대응

| 리스크 | 대응 |
|--------|------|
| STT 정확도 (한국어) | Whisper API 활용 + 텍스트 수정 UI 제공 |
| 발음 평가 정확도 | Azure Speech SDK 또는 검증된 API 사용 |
| OpenAI API 비용 | 일기 길이 제한 + 토큰 최적화 |
| TTS 자연스러움 | OpenAI TTS (높은 품질) 사용 |
